# Hospital-Efficiency-Performance-Analysis
This project focuses on measuring efficiency in hospital, which is one of the six dimensions in healthcare quality. 
Healthcare providers continually strives to reduce Length of Stay (LOS) and Costs where possible to maximize efficiency while not compromising the quality of patient care.

## Objective
- Analyse hospital performance using cost and LOS metrics  
- Identify variability across hospitals  
- Uncover key factors driving inefficiency  
- Provide actionable insights to support data-driven decision-making  

## Tools & Techniques
- Power BI (data modelling, DAX, dashboard design)  
- Data analysis and KPI development  
- Root cause analysis  
- Data visualisation and storytelling  

## Key Insights
- Significant variation exists in cost per discharge across hospitals  
- Patient severity and mortality risk are major drivers of cost and LOS  
- Urban hospitals exhibit higher cost and LOS trends  
- Discharge to skilled nursing facilities contributes to extended stays  

## Business Impact
- Enables benchmarking of hospital performance  
- Identifies inefficiency hotspots  
- Supports decision making for cost optimisation and resource allocation
